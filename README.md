## 📌 Tentang Aplikasi

**APOTEK HF** adalah sistem informasi manajemen apotek terintegrasi yang dirancang dengan antarmuka dasbor SaaS modern, responsif untuk layar komputer kasir maupun smartphone.

### ✨ Fitur Utama
- **🛒 Kasir POS (Point of Sale)**:
  - Pencarian obat instan dan dukungan multi-satuan (Satuan 1, 2, 3) dengan kalkulasi konversi otomatis.
  - Multi-level harga pelanggan (Level 1, Level 2, Level 3).
  - Cetak struk belanja thermal/digital & pemotongan stok otomatis.
- **💊 Master Obat & Stok Opname**:
  - Manajemen katalog obat lengkap dengan batas stok minimal.
  - Penyesuaian stok opname dan pemantauan obat menipis secara real-time.
- **🩺 Rekam Pelayanan Kesehatan**:
  - Pencatatan kontrol tensi darah, gula darah, asam urat, kolesterol, dan rekam medis pasien.
- **💰 Keuangan & Kas Operasional**:
  - Pembukuan mutasi kas masuk/keluar dan ringkasan omzet harian/bulanan.
- **🚚 Manajemen Supplier & Pelanggan**:
  - Pendataan mitra supplier, faktur pembelian restock, dan daftar pelanggan.

## 🛠️ Teknologi yang Digunakan

- **Frontend**: HTML5, Vanilla CSS3 (Custom SaaS Design System & Glassmorphism), JavaScript ES6+
- **Database**: Cloud PostgreSQL via **[Supabase](https://supabase.com)**
- **Hosting**: **[GitHub Pages](https://pages.github.com)** (Serverless)

---

## 🚀 Cara Menjalankan Secara Lokal

1. Clone repositori ini:
   ```bash
   git clone https://github.com/abro212/apotekhf.git
   cd apotekhf
   ```
2. Jalankan server lokal:
   ```bash
   npm start
   ```
3. Buka browser pada alamat `http://localhost:3000`.
